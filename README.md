### Example of creating a circular ProgressBar. 
Using NumberPicker to set the time.

![ProgressBar](https://github.com/user-attachments/assets/fbca56c3-ce35-4bfd-aa23-63b5c4b8026c)

### Attributes
|Attributes |Description| Related method(s)|Default value|
|-----------|-----------|-----------------|-------------|
|circleStroke|Thickness of the circle line| |0dp|
|dashColorActive|Active dash color|setDashColorActive |BLACK
|dashColorInactive|Color of inactive dash|setDashColorInactive|GRAY
|dashHeight|Dash height | |10dp
|dashRadius|Dash rounding radius| |0dp
|dashWidth|Dash width   | |3dp
|fontFamily|Font     |setFontFamily|DEFAULT
|max|Maximum value      |setMax, getMax     |20
|showPercent|Show percentages| |false
|text|Text           |setText|Empty|
|textColor|Text color| |BLACK
|textSize|Text size  | |14sp
|value|Current value |setValue, getValue|0
